# Personal Structured Knowledge Base

这是一个个人结构化知识库项目，目标规模约 1000 个知识节点。

## 项目结构

- `meta/`: 项目元信息、协作方式、命名规则与维护约定。
- `catalog.md`: 全局目录入口，用于索引知识库结构和重要入口。
- `knowledge/`: 真正的知识库内容区。当前保持为空，后续按节点逐步添加。

## 当前状态

- 框架已建立。
- 主知识库尚未添加任何知识节点。
- 后续新增、修改、移动或删除节点前，先提交变更方案供审阅；经你明确同意后再写入文件。

## 推荐使用方式

1. 你提出一个新增或修改需求。
2. 我先整理目标、影响范围和拟变更文件。
3. 你审阅并确认。
4. 我再写入变更，并说明这次改动如何组织知识。
